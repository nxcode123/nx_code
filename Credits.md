## 📦 Pihak Ketiga & Proyek Open-Source (Third-Party Software)
Skrip ini mengintegrasikan dan memanfaatkan teknologi luar biasa dari proyek-proyek berikut:
*   [Termux](https://github.com/termux/termux-app) — Emulator terminal dan lingkungan Linux yang kuat untuk Android.
*   [proot-distro](https://github.com/termux/proot-distro) — Penyedia utilitas *hypervisor* proot untuk menjalankan *rootfs* Ubuntu dengan aman tanpa akses root.
*   [Termux:X11](https://github.com/termux/termux-x11) — Server display grafis (X11) super lancar untuk menjembatani visual Linux ke layar Android.
*   [XFCE Desktop Environment](https://xfce.org/) — Lingkungan desktop Linux yang ringan, sangat stabil, dan ramah terhadap penggunaan memori perangkat seluler.
